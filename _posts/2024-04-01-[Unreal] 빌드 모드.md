## 개념 
+ 빌드 모드
	+ Debug	
		+ 언리얼 엔진, 프로젝트 **둘 다 Debug** 설정으로 빌드
	+ DebugGame
		+ **언리얼 엔진은 Release, 프로젝트는 Debug** 설정으로 빌드
	+ Development
		+ 언리얼 엔진, 프로젝트 **모두 Release** 설정으로 빌드
		+ 프로파일링 기능 등이 가능함
	+ Shipping
		+ 마찬가지로 언리얼 엔진, 프로젝트 모두 Release 설정으로 빌드
		+ **프로파일링 기능 등이 제거됨**
	+ Test
		+ 마찬가지로 모두 Release 설정으로 빌드됨
		+ QA를 위한 **일부 프로파일링 기능 등이 있음**
