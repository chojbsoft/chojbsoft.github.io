## Summary
+ 배치 파일: 윈도우에서 명령어를 자동화할 수 있는 파일
+ echo: 문자열 출력
+ echo.: 개행 출력
+ @echo off: 모든 명령어를 출력하지 않고 결과만 출력함
+ rem: 주석
+ set: 변수 정의
+ %변수명%: 변수 사용
+ %n: n번째 파라미터
+ call: 직접 실행과는 다르게, 다른 배치 파일을 실행한 뒤 다시 제어를 반환 받음
+ pause: 유저가 키를 누를 때 까지 일시정지
+ assoc: 파일 확장명과 파일 타입 연결
+ ftype: 특정 파일 타입을 실행할 때의 명령어 설정
